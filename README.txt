# Exile UI - Vietnamese Language Installer

Công cụ tự động cài đặt gói tiếng Việt hỗ trợ lvling cho Exile UI.

## Yêu cầu

- Python 3.6 trở lên https://www.python.org/ftp/python/3.14.3/python-3.14.3-amd64.exe
- Exile UI đã được tải về từ: https://github.com/Lailloken/Exile-UI

## Cách sử dụng

Chạy file run.bat

## Hướng dẫn thủ công

Copy /vi/[leveltracker] default guide.json vào thư mục data/english
Copy NotoSans-Regular.ttf vào thư mục /data/
Tìm alt_font : "Fontin-SmallCaps.ttf" trong Exile UI.ahk  thành alt_font := "NotoSans-Regular.ttf"
