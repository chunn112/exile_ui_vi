# Exile UI - Vietnamese Language Installer

Công cụ tự động cài đặt gói tiếng Việt cho Exile UI.

## Yêu cầu

- Python 3.6 trở lên
- Exile UI đã được tải về từ: https://github.com/Lailloken/Exile-UI

## Cách sử dụng

1. Copy file `installer.py` vào **thư mục gốc của Exile UI** (thư mục chứa file `Exile Ui.ahk`)
